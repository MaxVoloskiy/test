Задание к занятию «Сбор текстов из веба»

Преподаватель: Вероника Саркисян
Описание задания:

В этом домашнем задании вам предстоит:

    Выбрать свой источник структурированных данных (прогноз погоды, курс акций, счет спортивных мероприятий, или что-нибудь еще);
    Научиться извлекать данные из с выбранной веб-страницы;
    Научиться генерировать текст по шаблонам.

 
Часть 1

Выберете источник данных, который вам симпатичен. По умолчанию предлагаем использовать данные о погоде в Москве. Извлеките из html-кода страницы температуру сегодня, уровень осадков и скорость ветра.
 
Часть 2

Запрограммируйте шаблон, которые описывают температуру, уровень осадков и скорость ветра сегодня. Например, «Температура сегодня составляет X градусов», где Х — излеченное выше число.

Сгенерируйте таким образом описание погоды сегодня. Учтите, величина X влияет на грамматическое число, использованное в единицах измерения («двадцать градусов», но «двадцать один градус»).

Используйте бибилиотеку utg или дополнительные функции pymorphy2
In [1]:

import pymorphy2
